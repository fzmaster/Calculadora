Calculadora
===========

Calculadora feita em Java para o Curso de LP2 na FAESA 2014/2
Endereço Github: https://github.com/fzmaster/Calculadora

Funções implementadas por completo:
- Adição
- Subtração
- Multiplicação
- Divisão

Funções avançadas implementadas:
- Negação
- Potenciação
- Radiciação
- Logarítmo
